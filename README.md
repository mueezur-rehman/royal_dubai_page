## üìå Project Overview
**Royal Dubai** is a premium, high-end frontend landing page designed for an exclusive luxury jewelry and timepiece brand. ## ‚ú® Key Features Showcase (For Recruiters)
This project highlights several core frontend development skills:

* **Dynamic DOM Manipulation:** The "Signature Collections" section is not hardcoded; it dynamically renders data objects using JavaScript template literals, simulating a real-world API fetch scenario.
* **Intersection Observer API:** Implemented performant scroll animations (fade-in and slide-up effects) that trigger only when elements enter the viewport, optimizing rendering performance.
* **Modern CSS Architecture:** * Utilized **CSS Variables** (Custom Properties) for consistent color theming (Gold, Black, Glassmorphism).
    * Implemented **Glassmorphism** (`backdrop-filter`) for the Bespoke Form card to give it a modern, premium look.
    * Advanced **Keyframe Animations** (`fadeIn`, `slideUp`, `marquee`, `scrollAnimation`) for an interactive user experience.
* **Custom Interactive Components:** * A fully functional, auto-rotating **Testimonial Slider** built entirely from scratch with Vanilla JS.
    * A sticky navigation bar that adapts its styling upon scrolling.
* **Form Handling & Validation:** Prevented default form submission and simulated a backend request by dynamically generating a success UI state upon submission.
* **Fully Responsive Design:** Crafted with CSS Grid and Flexbox, ensuring a flawless layout from 4K desktop screens down to mobile devices (with a custom hamburger menu).
